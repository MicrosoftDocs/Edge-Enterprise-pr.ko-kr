# <span data-ttu-id="b25f8-103">Microsoft Edge 사용자 데이터 디렉터리 변수 만들기</span><span class="sxs-lookup"><span data-stu-id="b25f8-103">Create Microsoft Edge user data directory variables</span></span>

<span data-ttu-id="b25f8-107">데이터 디렉터리 경로를 수정하기 위한 정책(예: [UserDataDir](microsoft-edge-policies.md#userdatadir) 또는 [DownloadDirectory](microsoft-edge-policies.md#downloaddirectory) 지원 변수 구성)</span><span class="sxs-lookup"><span data-stu-id="b25f8-107">Policies for modifying data directory paths (For example, configuring the [UserDataDir](microsoft-edge-policies.md#userdatadir) or [DownloadDirectory](microsoft-edge-policies.md#downloaddirectory) support variables.</span></span> <span data-ttu-id="b25f8-108">이러한 정책을 구성할 때 하드 코드된 경로 대신 변수를 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="b25f8-108">When configuring these policies, you can use variables instead of hard-coded paths.</span></span> <span data-ttu-id="b25f8-109">예를 들어 프로필 데이터를 기본 위치 대신 Windows의 사용자 로컬 응용 프로그램 데이터 아래에 저장합니다.</span><span class="sxs-lookup"><span data-stu-id="b25f8-109">For example, to store your profile data under user local application data on Windows instead of the default location.</span></span> <span data-ttu-id="b25f8-110">[UserDataDir](microsoft-edge-policies.md#userdatadir) 정책을 **${local_app_data}\Edge\Profile**로 설정합니다.</span><span class="sxs-lookup"><span data-stu-id="b25f8-110">Set the [UserDataDir](microsoft-edge-policies.md#userdatadir) policy to **${local_app_data}\Edge\Profile**.</span></span> <span data-ttu-id="b25f8-111">대부분의 Windows 10 설치에서 이는 *C:\Users\\&lt;Current-user&gt;\AppData\Local\Microsoft\Edge\Profile*로 확인됩니다.</span><span class="sxs-lookup"><span data-stu-id="b25f8-111">On most Windows 10 installations, this path resolves to *C:\Users\\&lt;Current-user&gt;\AppData\Local\Microsoft\Edge\Profile*.</span></span>

>[!NOTE]
><span data-ttu-id="b25f8-112">이 정책이 설정되지 않으면 Microsoft Edge가 기본 디렉터리 경로를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="b25f8-112">If left unset, Microsoft Edge uses default directory paths.</span></span> <span data-ttu-id="b25f8-113">사용자는 `--user-data-dir` 명령줄 플래그를 사용하여 기본값을 재정의할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="b25f8-113">A user can use the `--user-data-dir` command-line flag to override the default.</span></span>

- <span data-ttu-id="b25f8-118">Microsoft Edge가 다양한 데이터를 저장하는 경로를 포함하는 모든 정책은 플랫폼마다 다릅니다.</span><span class="sxs-lookup"><span data-stu-id="b25f8-118">All policies that involve paths where Microsoft Edge stores different data are platform dependent.</span></span> <span data-ttu-id="b25f8-119">이러한 정책 중 일부는 특정 플랫폼에서만 사용할 수 있지만 다른 정책은 모든 플랫폼에서 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="b25f8-119">Some of these policies are available only on specific platforms, but others can be used on all platforms.</span></span>
- <span data-ttu-id="b25f8-120">응용 프로그램이 경우마다 다른 위치에서 시작하여 발생하는 오류를 방지하려면 절대 경로를 설정해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="b25f8-120">To avoid errors caused by applications starting from different locations on different occasions, make sure that paths are absolute.</span></span>
